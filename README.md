# NomeProgettoUE5

Repository del progetto Unreal Engine 5 ABABO.

## Tool usati
- Unreal Engine 5
- GitHub Desktop
- GitHub
- Git LFS

## Regole di lavoro
1. Fai `Pull` prima di aprire il progetto in Unreal.
2. Non lavorare in due sulla stessa mappa `.umap`.
3. Fai commit piccoli e con messaggi chiari.
4. Fai `Push` a fine sessione.

## Workflow studenti
1. Apri GitHub Desktop
2. Fai `Fetch origin` / `Pull origin`
4. Apri il progetto UE5
5. Lavora
6. Salva
7. Commit
8. Push

Esempi:
- `feature/mario-rossi`
- `feature/player-movement`
- `fix/door-collision`

## Naming commit
Formato consigliato:

`[Area] descrizione breve`

Esempi:
- `[UI] aggiunto menu pausa`
- `[Player] fix dash`
- `[Level] sistemata collisione stanza 01`
